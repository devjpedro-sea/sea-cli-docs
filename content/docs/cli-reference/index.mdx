## Comandos Globais

```bash
sea-cli --version    # Exibe a versão instalada
sea-cli --help       # Mostra ajuda geral
sea-cli <comando> --help  # Ajuda de um comando específico
```

## Início Rápido

Para começar rapidamente, use:

```bash
sea-cli init
```

Este comando abre um menu interativo onde você pode escolher a ação desejada.
