# Introdução ao SEA CLI

O **SEA CLI** é uma ferramenta de linha de comando desenvolvida pela **SEA Tecnologia** que revoluciona a forma como você configura padrões de qualidade de código em projetos Liferay DXP.

## O que é o SEA CLI?

SEA CLI é uma ferramenta de automação que simplifica drasticamente a configuração de padrões de qualidade de código em projetos Liferay. Ele automatiza tarefas que normalmente levariam **horas** para serem configuradas manualmente.

<Callout type="info">
  **Economize tempo!** Configure ESLint, Prettier, Husky, Jest e muito mais em
  minutos, não em horas.
</Callout>

## Por que usar?

### Sem SEA CLI

- Horas configurando ferramentas manualmente
- Configurações inconsistentes entre projetos
- Risco de erros humanos
- Onboarding demorado de novos desenvolvedores
- Falta de padronização

### Com SEA CLI

- Configuração automática em minutos
- Padronização garantida em todos os projetos
- Configurações testadas e validadas
- Onboarding rápido e eficiente
- Qualidade de código consistente

## O que é configurado automaticamente?

### ESLint + Prettier

- Regras pré-configuradas para React e JavaScript moderno
- Formatação automática de código
- Integração perfeita entre as ferramentas

### Husky + lint-staged

- Validação automática antes de commits
- Impede commits com código problemático
- Feedback imediato sobre problemas

### Jest + Testing Library

- Framework de testes completo
- Testes de componentes React
- Cobertura de código automática

### Scripts de Automação

- Build e deploy para Liferay
- Lint e formatação
- Execução de testes
- Geração de relatórios
