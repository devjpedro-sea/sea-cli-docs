# SEA CLI - Automação de Qualidade para Liferay

O **SEA CLI** é uma ferramenta de linha de comando desenvolvida pela **SEA Tecnologia** que automatiza a configuração de padrões de qualidade de código em projetos Liferay DXP.

## Por que usar o SEA CLI?

<Callout type="info">
  **Economize horas de configuração!** O que levaria horas para configurar
  manualmente, o SEA CLI faz em minutos.
</Callout>

### Principais Benefícios

- **Automação Completa**: Configuração de ESLint, Prettier, Husky, Jest e mais
- **Produtividade**: Scripts prontos para build, deploy e testes
- **Padronização**: Garante consistência em todos os projetos
- **Testes Integrados**: Jest + Testing Library pré-configurados
- **Qualidade**: Validação automática antes de cada commit

## Casos de Uso

<Steps>
### Criar novo workspace Liferay
Clone e configure um workspace completo em um único comando

### Adicionar qualidade em projeto existente

Instale todas as configurações em workspaces já criados

### Criar módulos pré-configurados

Gere módulos com lint, testes e scripts prontos para uso

### Configurar Turborepo

Otimize monorepos com cache inteligente de builds

</Steps>
